6502 Firmware
=============

This folder contains firmware for the 6502 project.

Folder names are prefixed with the following to indicate specific hardware environments:

- **DB**: Dev Board
- **DOB**: Dev Output Board
- **IB**: Input Board
- **KEH**: Keyboard Encoder Helper
- **OH**: Output Helper
- **PS2**: PS2 Helper
- **SCP**: Storage Card Pro
- **VGAP**: VGA Card Pro
- **VCP**: Video Card Pro
