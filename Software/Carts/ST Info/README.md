ST Info
=======

Prints Compact Flash card info

Adapted for the 6502 from: [https://www.waveguide.se/?article=8-bit-compact-flash-interface](https://www.waveguide.se/?article=8-bit-compact-flash-interface)

Install:

    brew install cc65

Build:

    make

View:

    make view

Burn EEPROM:

    make eeprom
